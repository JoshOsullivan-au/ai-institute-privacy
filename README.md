# AI Institute Privacy Policy

This repository hosts the privacy policy for the AI Institute Vector Store Integration service.

## Files

- `index.html` - Main privacy policy page (GitHub Pages)
- `privacy_policy.md` - Markdown version
- `PRIVACY_POLICY_DEPLOYMENT.md` - Deployment guide

## Quick Access

Once GitHub Pages is enabled, your privacy policy will be available at:
`https://yourusername.github.io/ai-institute-privacy/`

## Setup Instructions

1. Create GitHub repository: `ai-institute-privacy`
2. Upload these files
3. Enable GitHub Pages in repository settings
4. Set source to main branch, root folder
5. Your privacy policy URL will be ready!

## Custom GPT Integration

Use this URL in your Custom GPT configuration for the privacy policy field.
